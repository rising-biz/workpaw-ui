# WorkPaw Desktop 重新设计规格

> 日期: 2026-07-03 | 状态: 已确认 | 替换: `docs/optimization-plan-desktop.md` (旧版含已废弃的 DAG 编排 + Coding Mode)

---

## 一、产品定位

**WorkPaw Desktop = 企业级 QwenPaw 桌面客户端**

三层价值:
- **C (底座)**: QwenPaw 能力完整移植到桌面端，Web 端能做的 Desktop 都能做
- **A (安全差异化)**: 企业安全可见性——策略透明、审批可见、操作可审计。让 IT 敢放行，让员工放心用
- **B (降低门槛)**: 桌面原生体验——快捷键、通知、拖拽、场景画廊。30 秒从安装到第一次成功对话

**已明确砍掉**: DAG 工作流编排（用户是员工不是 Agent 开发者）、Coding Mode（开发者有专业 IDE）

---

## 二、阶段 C：底座对齐 — QwenPaw Web Chat 能力完整移植

### C.1 消息队列

- 多消息排队发送，Tauri 原生快捷键 (`Ctrl+Enter` 发送, `Shift+Enter` 换行)
- 拖拽排序 (`@dnd-kit`，复用 QwenPaw 已有组件)
- 暂停/恢复/全部发送
- 编辑队列中消息，删除队列中消息
- 不需要跨标签页同步（单窗口桌面应用）

### C.2 审批卡片

- Tool Guard 审批卡片在 Chat 流中内嵌渲染
- Approve/Deny + Pattern 范围 + 作用域选项（复用 QwenPaw 已有组件）
- Desktop 增强: 审批操作触觉反馈（系统通知确认），审批历史关联对应消息

### C.3 Slash Commands

- 输入 `/` 触发自动补全面板
- 支持: `/clear`, `/compact`, `/mission`, `/goal` + Skills 自动补全
- 支持插件注册的自定义命令

### C.4 Model Selector

- 输入区顶部或侧边显示当前模型
- 点击弹出模型列表（复用 QwenPaw 已有组件）
- OAuth 确认流程

### C.5 文件上传

- 多模态附件支持
- 粘贴图片/文件自动上传
- 拖拽文件到 Chat 窗口
- 上传前模型兼容性检测提示

### C.6 消息编辑与重试

- 编辑已发送消息后重新发送
- ↑ 键回溯上一条消息
- 失败消息右键重试

### C.7 Session 管理增强

- 会话列表（搜索/置顶/重命名）
- 历史面板（右侧抽屉）
- 会话分组（今天/昨天/更早）

### C.8 Loop 模式状态可视化

- Goal Mode / Mission Mode 运行时顶部状态条
- 显示: 当前阶段、完成进度、剩余任务数、已消耗 Token、运行时间
- 支持暂停/停止

## 三、阶段 A：企业安全差异化

### A.1 策略可见性面板

- Chat 侧边栏或顶部显示当前用户权限边界
- 三色标识: 🟢 允许 / 🟡 需审批 / 🔴 禁止
- 每项悬停显示限制原因 tooltip
- 最近被拦截的操作高亮闪烁
- 默认折叠，Agent 被拦截时自动展开
- 策略来源标注（策略名称 + 管理员联系方式）

### A.2 审批流内嵌 Chat

- 审批卡片在对话流中内嵌，显示上下文:
  - 操作目标（文件路径/命令/API）
  - 影响范围（文件数/数据量/受影响系统）
  - 风险级别（中危黄色/高危红色）
- 审批选项: 批准 / 拒绝 / 记住选择 / 作用域限定
- 审批后卡片折叠为一行摘要
- 审批依据可展开查看

### A.3 操作审计回放（员工视角）

- 员工自己的操作历史列表，按 Session 分组
- 每步操作: 图标 + 描述 + 状态（通过/审批/拒绝）+ 时间
- 回放模式: 逐步骤动画展示操作过程
- 统计: Token 消耗、耗时
- 搜索 + 日期筛选

### A.4 Session 安全标识

- Chat 顶部紧凑状态条
- 显示: 沙箱状态 / 连接的外部 MCP 服务 / 传输加密状态 / 数据区域
- 外部 MCP 用橙色标识
- 默认一行，点击展开详情

### A.5 敏感操作二次确认

- 危险操作前弹窗，展示:
  - 操作描述 + 影响范围
  - 风险提示（不可恢复/影响生产/涉及外部）
  - 文件最后修改时间（删除操作时）
- 确认/取消按钮

### A.6 沙盒模式开关

- 手动切换开关
- 开启后 Agent 操作限制在 `/tmp/workpaw-sandbox/{session_id}/`
- 新手默认开启
- 切换时说明后果

### A.7 数据外传提醒

- 当 Agent 将数据发送到企业白名单外的外部服务时触发
- 底部滑入提醒，3 秒自动收起
- 显示: 目标服务器地址、数据量、内容摘要
- 非阻断式

### A.8 紧急锁定

- `Cmd+Shift+L` 全局快捷键
- 立即冻结 Agent 所有操作
- 显示: 锁定时间、已中断操作
- 托盘图标变红
- 解锁需要确认

### A.9 Agent 能力边界卡片

- 首次使用某 Agent 时自动展示
- 列出: 能做什么（文件目录/工具/服务）+ 不能做什么
- "开始对话" 按钮直接进入 Chat
- 后续可在 Agent 设置中重新查看

### A.10 Token/成本感知

- Chat 底部状态栏轻量显示: 本次消耗 + 本月累计
- 接近月度预算上限时变橙色
- 支持隐藏或切换为 Token 模式
- 价格按模型实际成本换算

### A.11 策略反馈通道

- 被策略拦截时展示申请入口
- 填写申请理由 → 提交给管理员
- 追踪申请状态
- 与 Admin 端审批工作流打通

## 四、阶段 B：降低门槛

### B.1 全局快捷键唤起

- `Cmd+Shift+A` 呼出浮动对话窗口
- 半透明毛玻璃背景，悬浮在当前桌面/应用之上
- 输入问题直接发送给默认 Agent
- 支持 `/` 命令、`@` 切换 Agent
- 窗口大小可拖拽，记住位置
- Esc 关闭；结果出来后系统通知

### B.2 系统原生通知

- macOS 通知中心 + Windows 通知栏
- 审批通知支持内嵌操作（Approve/Deny）按钮
- 任务完成通知可直接跳转 Session
- 同 Session 多条通知自动折叠
- 勿扰模式（工作时间外静默，次日汇总）

### B.3 文件拖拽交互

- 文件/图片拖入 Chat 窗口 → 自动上传/多模态输入
- 多文件拖入 → 自动打包为消息队列
- Agent 生成文件可拖出到桌面/Finder
- 截图 (`Cmd+Shift+4`) → 粘贴 (`Cmd+V`) → Agent 自动分析
- 大文件显示进度条

### B.4 场景画廊增强 — "一键上岗"

- 基于角色推荐场景（使用次数标注）
- "一键上岗"流程: 点击卡片 → 自动配置 Agent + Skills + MCP + Prompt → 10 秒内进入 Chat
- 第一句话预填
- 企业专属分类（管理员推送的模板）
- 场景搜索 + 分类浏览
- "请求新场景"交互

### B.5 右键上下文菜单集成

- 通过 Tauri 注册 macOS Services / Windows Context Menu
- 选中文件 → "发给 Agent 分析" / "让 Agent 总结" / "让 Agent 翻译"
- 可选目标 Agent
- 多文件批量操作
- 操作: 打开 WorkPaw → 上传文件 → 附 prompt → Agent 开始处理

### B.6 托盘常驻模式

- 主窗口关闭 → 自动缩到系统托盘
- Agent 后台继续工作
- 托盘图标动画: 静止(空闲) / 呼吸(思考中) / 脉冲(等待审批) / 红色(异常)
- 任务完成时气泡提醒
- 右键菜单: 进度 / 暂停 / 打开 Chat / 设置 / 退出

### B.7 自然语言 → Agent 配置

- 用户用自然语言描述需求
- Agent 解析 → 匹配场景模板 → 生成 Agent + Skills + MCP + Cron 配置
- 生成结果可预览和手动微调
- "一键创建" 完成全部配置
- 降低用户学习成本（不需理解 Agent/Skill/MCP/Cron 概念）

### B.8 新用户引导

- 3 步引导流程:
  1. 选择岗位角色
  2. 勾选推荐的 Agent 场景
  3. 进入第一次对话
- 全程可跳过
- 之后可在设置中重新运行或切换角色

---

## 五、不出现在 Desktop 的功能

以下能力保留在 Web 端，Desktop 不重复实现:

- Agent 详细配置管理
- Skills/MCP/Models/Channels 管理
- 全局 Settings
- 管理员功能（使用 Admin Console）
